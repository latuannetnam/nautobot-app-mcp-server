## 09-05: Lazy Import Audit

### Context

`query_utils.py` has module-level Nautobot model imports on lines 16-17:
```python
from nautobot.dcim.models import Device, Interface, Location
from nautobot.ipam.models import VLAN, IPAddress, Prefix
```

These are the only violations in `mcp/tools/`. The conversion pattern: move these imports to a `TYPE_CHECKING` block (for type annotations) and add lazy imports inside each function that uses them.
