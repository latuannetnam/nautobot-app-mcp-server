# Plan 02: Core Read Tools

<objective>
Implement all 10 core read tools (`device_list`, `device_get`, `interface_list`, `interface_get`, `ipaddress_list`, `ipaddress_get`, `prefix_list`, `vlan_list`, `location_list`, `search_by_name`). Each tool: uses `sync_to_async` with `thread_sensitive=True`, calls `.restrict(user, "view")`, returns a `PaginatedResult` or typed dict, and is registered at module level with `scope="core"`.
</objective>

## Decisions

- **Scope for all 10 tools:** `"core"` — always visible, bypass progressive disclosure (D-04, SESS-06)
- **Tier for all 10 tools:** `"core"` — always-available tools
- **Not-found behavior (D-02):** `device_get`, `interface_get`, `ipaddress_get` raise `ValueError` with message `"Device 'X' not found"` etc. Do NOT return null.
- **Identifier lookup (D-03):** Single-object tools accept `name_or_id: str`; if it looks like a UUID (32 hex chars with hyphens), do `pk` lookup; otherwise do `name` lookup
- **Serializer approach:** `model_to_dict()` with `fields`/`exclude` for flat fields; inline FK flattening via `display_name`/`name` attribute access; recursive serialization for prefetched reverse relations
- **All ORM via `sync_to_async(fn, thread_sensitive=True)`:** no ORM calls in async handlers directly (PAGE-05)

## Must-Haves

- All 10 tools return data; all list tools return `PaginatedResult`; all single-object tools return a dict or raise `ValueError`
- Every tool calls `.restrict(user, action="view")` on its queryset
- `device_get`: `interfaces` prefetched and serialized inline
- `interface_get`: `ip_addresses` prefetched and serialized inline
- `ipaddress_get`: `interfaces` prefetched and serialized inline
- All 10 tools registered at module level with `register_mcp_tool(scope="core", tier="core")`

```python
def serialize_device(device: Device) -> dict[str, Any]:
    """Serialize a Device to a JSON-serializable dict."""
    data = model_to_dict(
        device,
        fields=[
            "pk", "name", "serial", "asset_tag", "description",
            "primary_ip4", "primary_ip6",
        ],
        exclude=_STANDARD_EXCLUDE,
    )
    # Flatten FK objects inline
    data["status"] = device.status.name if device.status else None
    data["role"] = device.role.name if device.role else None
    data["platform"] = device.platform.name if device.platform else None
    data["device_type"] = device.device_type.display_name if device.device_type else None
    data["manufacturer"] = (
        device.device_type.manufacturer.name if device.device_type and device.device_type.manufacturer else None
    )
    data["location"] = device.location.name if device.location else None
    data["tenant"] = device.tenant.name if device.tenant else None
    data["pk"] = str(device.pk)
    return data


def serialize_device_with_interfaces(device: Device) -> dict[str, Any]:
    """Serialize a Device with its interfaces prefetched."""
    data = serialize_device(device)
    # Interfaces are prefetched in the queryset; serialize inline
    if hasattr(device, "_prefetched_objects_cache") and "interfaces" in device._prefetched_objects_cache:
        data["interfaces"] = [
            serialize_interface(iface) for iface in device.interfaces.all()
        ]
    else:
        data["interfaces"] = [
            serialize_interface(iface) for iface in device.interfaces.all()
        ]
    return data
```

```python
def _sync_device_list(user: User, limit: int, cursor: str | None) -> dict[str, Any]:
    """Sync implementation of device_list."""
    from nautobot_app_mcp_server.mcp.tools.pagination import paginate_queryset
    qs = build_device_qs().restrict(user, action="view")
    result = paginate_queryset(qs, limit=limit, cursor=cursor)
    return {
        "items": [serialize_device(d) for d in result.items],
        "cursor": result.cursor,
        "total_count": result.total_count,
        "summary": result.summary,
    }


def _sync_device_get(user: User, name_or_id: str) -> dict[str, Any]:
    """Sync implementation of device_get (D-02 not-found + D-03 name-or-id)."""
    from nautobot_app_mcp_server.mcp.tools.pagination import encode_cursor
    # D-03: UUID-like → pk lookup, otherwise → name lookup
    if _looks_like_uuid(name_or_id):
        qs = build_device_qs().filter(pk=name_or_id).restrict(user, action="view")
    else:
        qs = build_device_qs().filter(name=name_or_id).restrict(user, action="view")
    # Prefetch interfaces for the single device
    qs = qs.prefetch_related("interfaces__ip_addresses", "interfaces__status")
    devices = list(qs)
    if not devices:
        raise ValueError(f"Device '{name_or_id}' not found")
    return serialize_device_with_interfaces(devices[0])


def _looks_like_uuid(value: str) -> bool:
    """Return True if value looks like a UUID (D-03 identifier auto-detection)."""
    import re
    return bool(re.match(r"^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$", value.lower()))
```
