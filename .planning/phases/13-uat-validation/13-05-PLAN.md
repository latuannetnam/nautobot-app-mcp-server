### If tests fail

| Test failure | Likely cause | Fix |
|-------------|-------------|-----|
| `ConnectionError` on `localhost:8005` | mcp-server not running | `invoke start`; check `docker ps` for mcp-server |
| T-01 fails: missing tools | `register_all_tools_with_mcp()` not called | Verify `create_app()` imports `core` module (Phase 9) |
| T-27 fails: not empty | Auth token lookup broken | Check `get_user_from_request()` in `auth.py` (Phase 11) |
| T-01 fails: only 3 tools | FastMCP state API broken | Check `session_tools.py` `_list_tools_handler` (Phase 10) |
| All tests SKIP | No data imported | `poetry run invoke import-data` or `bash scripts/reset_dev_db.sh --import` |
| HTTP 500 from MCP | FastMCP server error | `docker logs mcp-server-1` for traceback |
