### Verify MCP server is reachable

```bash
# 7. Check port 8005 is listening
curl -s -o /dev/null -w "%{http_code}" http://localhost:8005/mcp/
# Expected: 200 or 405 (MCP endpoint alive; GET returns 405 "method not allowed" which is correct)

# 8. Verify container logs show FastMCP startup
docker logs nautobot-app-mcp-server-mcp-server-1 --tail 20
# Expected: "[start_mcp_dev_server] Starting FastMCP dev server..." followed by uvicorn binding to 127.0.0.1:8005
# Note: start_mcp_dev_server binds to 127.0.0.1:8005 inside the container, mapped to 0.0.0.0:8005 on host

# 9. Confirm old endpoint returns 404 (Phase 12 gate)
curl -s -o /dev/null -w "%{http_code}" http://localhost:8080/plugins/nautobot-app-mcp-server/mcp/
# Expected: 404 (Phase 12 deleted urls.py entry)
```

### Container verification checklist

| Check | Command | Expected |
|-------|---------|----------|
| `mcp-server` container exists | `docker ps -q --filter "name=mcp-server"` | non-empty |
| `mcp-server` running | `docker inspect --format '{{.State.Status}}' nautobot-app-mcp-server-mcp-server-1` | `running` |
| Port 8005 open | `curl -s -o /dev/null -w "%{http_code}" http://localhost:8005/mcp/` | `200` or `405` |
| Old endpoint 404 | `curl -s -o /dev/null -w "%{http_code}" http://localhost:8080/plugins/nautobot-app-mcp-server/mcp/` | `404` |
| MCP server logs clean | `docker logs nautobot-app-mcp-server-mcp-server-1 --tail 5` | no exceptions |

### <acceptance_criteria>

- [ ] `docker ps` shows `nautobot-app-mcp-server-mcp-server-1` as `running`
- [ ] `curl http://localhost:8005/mcp/` returns HTTP 200 or 405 (not 404, not connection refused)
- [ ] `curl http://localhost:8080/plugins/nautobot-app-mcp-server/mcp/` returns HTTP 404 (Phase 12 gate)
- [ ] Container logs show no `RuntimeError` from `create_app()` (DB connectivity check passes)
- [ ] `docker logs mcp-server-1 --tail 20` shows FastMCP startup messages

### If MCP server fails to start

| Error | Likely cause | Fix |
|-------|-------------|-----|
| `nautobot-server: command not found` | Image built before Phase 8/9/10/11/12 changes | `poetry run invoke build && poetry run invoke start` |
| `RuntimeError: Database connectivity check failed` | DB not ready / env vars missing | Check `NAUTOBOT_DB_*` vars in mcp-server container |
| `Connection refused` on port 8005 | MCP server crashed or didn't bind | `docker logs mcp-server-1` to see traceback |
| Port already in use | Another process on 8005 | `lsof -i :8005` to identify conflict |
