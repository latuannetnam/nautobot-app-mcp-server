# Plan 11-02: Update call sites + nginx documentation

**Phase:** 11-auth-refactor · Wave 2
**Requirements:** P4-03, P4-04
**Files touched:** `nautobot_app_mcp_server/mcp/tools/core.py`, `docs/admin/upgrade.md`

---

## Context

Wave 1 (Plan 11-01) refactored `get_user_from_request()` to `async def`. All 10 call sites in `core.py` must now `await` it. The downstream `.restrict(user, "view")` calls in `query_utils.py` are **not modified** — the user object is still returned identically.

nginx strips `Authorization` headers by default (RFC 7230). Without `proxy_set_header Authorization $http_authorization;`, all MCP requests arrive as `AnonymousUser`. This must be documented in `upgrade.md`.

## Task 11-02-02: Document nginx `proxy_set_header Authorization` in `upgrade.md`

**Steps:**

1. Read `docs/admin/upgrade.md` to confirm current content structure.
